NamesFromEmail generiert aus einer Email-Adresse den Vornamen und Namen des Besitzers.

Das Programm erwartet eine CSV-Datei. Die Spalte mit den Emails muss zur Zeit die Erste sein.
Um das Programm auszufuehren muessen folgende Parameter angegeben werden:

1. InputFile.csv
2. OutputFile.csv
3. Delimiter umschlossen von Hochkomma

Zusaetzlich koennen Standardvor- und Standardnachname angegeben werden, falls keine Name ermittelt werden kann:

4. Standardvorname
5. Standardname
